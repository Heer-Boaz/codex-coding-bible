# Codex Coding Bible

Personal Codex marketplace containing the `heer-boaz-coding-bible` plugin.

## Install

Add this marketplace to Codex:

```bash
codex plugin marketplace add https://github.com/Heer-Boaz/codex-coding-bible.git
```

The marketplace installs the plugin from:

```text
plugins/heer-boaz-coding-bible
```

## Update

Refresh the installed marketplace:

```bash
codex plugin marketplace upgrade codex-coding-bible
```

## Contents

- `feature-code`: feature and bug-fix implementation rules.
- `lean-code`: cleanup and architecture repair rules.
- `quality-analysis`: code-quality analyzer and lint workflow rules.
- `format-fix`: formatting-only repair workflow.
